This project is a starter template and a full example to recreate a provided LG design with your own branding.

## Quick Start

1. Install dependencies:
   ```
   npm install
   ```
2. Start dev server:
   ```
   npm run dev
   ```

## Features
- Vite + React
- Tailwind CSS with branded palette
- Routing (React Router)
- Dark mode toggle with persistence
- Reusable components and pages
- Responsive layout (mobile/tablet/desktop)

## Structure
See `/src` for components, layouts, pages, and assets.

## Deployment
Deploy to Vercel or Netlify. Build with `npm run build`.
